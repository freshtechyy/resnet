# PyTorch Implementation for ResNets

This repo shows a PyTorch implementation for ResNets (Residual Networks) $^{[1]}$ from sratch.

## Dependencies
* Python 3.8.10
* PyTorch 1.9.1

## Usages
```bash
$ git clone 
$ cd resnet/
$ python3 resnet.py
```

## References
* K. He, et. al, "Deep Residual Learning for Image Recognition", CVPR, 2016.# resnet
